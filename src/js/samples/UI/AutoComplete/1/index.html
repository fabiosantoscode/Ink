AutoComplete Samples - InkJS

Intro and basic concepts

Ink.UI.AutoComplete is for creating autocomplete suggestions in your input elements, which are relevant to your application.

You need to give it some suggestions, or a way to get suggestions. There are a couple of ways to do this:

  • Passing it an Array of everything which could be typed in options.suggestions. This is the simplest way to do this, but you need to have a list of every valid input around
  • Passing a URI for a JSON endpoint which responds with suggestions. You can use the suggestionsURI, suggestionsURIParam, and the getSuggestionsURI options.
